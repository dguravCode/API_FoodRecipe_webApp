# Food Recipe API Project

A dynamic web application that fetches and displays food recipes using the TheMealDB api API. Search for your favorite recipes according to different country.

## Features

- **Recipe Search**: Search for recipes by ingredients or dish name
- **Recipe Details**: View comprehensive recipe information including:
  - Meal Name.
  - Meal Representation. 
  - Filter Recipe according to Country.
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Real-time API Integration**: Fetches live data from TheMealDB API

## Getting Started

### Prerequisites
- Modern web browser
- Internet connection for API calls

### Installation

1. Clone or download this project
2. Open `index.html` in your web browser
3. Start searching for recipes!

## How to Use

1. Enter an ingredient or recipe name in the search box
2. Click or press 'Enter'
3. Browse through the recipe results
4. See Visual Representation of versatile Recipe.

## API Reference

This project uses the [TheMealDB Recipe API](https://www.themealdb.com/api.php) to fetch recipe data.
